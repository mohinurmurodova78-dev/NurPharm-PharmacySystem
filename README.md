# NurPharm - Система управления аптечным складом

## Описание
NurPharm — это современное приложение для автоматизации учёта лекарственных препаратов в аптеке. Система помогает контролировать остатки, сроки годности и быстро находить нужные препараты.

## Функционал
- 🔍 Удобный поиск по названию, категории
- 📅 Контроль сроков годности
- 💊 Учёт рецептурных и безрецептурных препаратов
- 📊 Сортировка по цене, количеству, сроку годности
- 💾 Хранение данных в XML-формате

## Технологии
